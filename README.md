# placetrump
A scary image placeholder platform using Donald Trump images.

## Requirements

- Node.js version 5
- [libvips](https://github.com/jcupitt/libvips)

## Usage

Install the dependencies:
```
npm install -g gulp
npm install
bower install
```

Download some Donald Trump images:
```
gulp fetch-images
```

Run!
```
npm start
```
